# autumn-cloud

#### 介绍
autumn-cloud 开始编写，致力于以最简单的代码（🈲过度封装）。

#### 软件架构
软件架构说明
SpringBoot, SpringCloud AliBaBa, Mybatis-Plus, Mysql, Redis, sa-token

#### 安装教程

1.  安装nacos
2.  安装redis
3.  安装mysql
4.  推荐使用docker

#### 使用说明

1.  更改nacos动态配置的各种地址
2.  更改代码配置文件的各种地址
3.  运行gateway, system
