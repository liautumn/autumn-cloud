# autumn-cloud

#### 介绍
autumn-cloud 编写代码中，致力于以最简单的代码（🈲过度封装）学习成本极低。

#### 软件架构
框架选型：
SpringBoot, SpringCloud AliBaBa, Mybatis-Plus, Mysql, Redis, sa-token, redisson, seata, ShardingSphere, Hutool, mysql, minio, rabbitmq 等

#### 安装教程

1.  安装mysql
2.  安装redis
3.  安装nacos
3.  安装minio
4.  推荐使用docker

#### 使用说明

1.  更改nacos动态配置的各种地址
2.  更改代码父Maven配置，选择环境
3.  运行gateway, auth, system 启动
